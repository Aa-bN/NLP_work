## 模型功能
A. 完成基本的LSTM过程  
B. 可以选择LSTM的层数，通过传入参数，实现n(int, ≥1)层LSTM的使用  
C. 可以选择性地传入hidden state, cell state, batch first等参数  
D. 输出outputs，每层最终的hidden state, cell state  
E. 对于模型输入输出的tensor数据，规范了其shape
